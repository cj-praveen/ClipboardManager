### Build from Source
```
git clone https://github.com/cj-praveen/ClipboardManager.git
cd ClipboardManager
python3 -m venv .
source bin/activate
pip3 install -U pip
pip3 install -r requirements.txt py2app
python3 setup.py py2app
open dist
```
